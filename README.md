# To-Do Web Application

SkillCraft Technology - Web Development Internship

## Task 04: To-Do Web Application

### Project Description

This project was developed as part of **Task 04** of the **SkillCraft Technology Web Development Internship**.

The objective of this task was to design and develop a To-Do Web Application using **HTML, CSS, and JavaScript**. The application allows users to create, manage, and organize daily tasks by adding new tasks, marking them as completed, and removing tasks when they are no longer needed.

### Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Interactive and user-friendly interface
* Responsive design for desktop and mobile devices
* Clean and modern user interface

### Technologies Used

1. HTML5
2. CSS3
3. JavaScript
